动态内存

malloc/free 堆内存

📌 学习目标

掌握 malloc/free 堆内存，通过一个可运行的小程序理解核心用法。

知识点

  • malloc(字节数) 申请堆内存，返回 void* 需要强转
  • 用完必须 free，否则内存泄漏
  • realloc 调整大小，calloc 申请并清零
⚠️ 易错点
  • 忘记 free 导致内存泄漏
  • free 后继续使用指针（悬空指针）是严重错误
  • malloc 返回 NULL 表示失败，大内存申请要检查
💡 课后练习

用动态数组实现一个能自动扩容的 int 容器。

main.c

#include <stdio.h>
#include <stdlib.h>

/* 第 14 天：动态内存
 * 用 malloc 创建运行时才知大小的数组。
 */
int main(void) {
    int n = 5;

    /* 动态分配 n 个 int */
    int *arr = malloc(n * sizeof(int));
    if (arr == NULL) {
        printf("内存分配失败\n");
        return 1;
    }

    /* 赋值 */
    for (int i = 0; i < n; i++) {
        arr[i] = (i + 1) * (i + 1);  /* 平方数 */
    }
    printf("初始数组：");
    for (int i = 0; i < n; i++) printf(" %d", arr[i]);
    printf("\n");

    /* 扩容到 8 */
    int *bigger = realloc(arr, 8 * sizeof(int));
    if (bigger == NULL) {
        free(arr);
        return 1;
    }
    arr = bigger;
    for (int i = n; i < 8; i++) arr[i] = (i + 1) * (i + 1);
    printf("扩容后：");
    for (int i = 0; i < 8; i++) printf(" %d", arr[i]);
    printf("\n");

    /* 必须释放 */
    free(arr);
    printf("内存已释放\n");
    return 0;
}

▶ 运行结果

初始数组： 1 4 9 16 25
扩容后： 1 4 9 16 25 36 49 64
内存已释放

cd c-30days/day14
