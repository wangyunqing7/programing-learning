条件判断

if / else if / else 分支

📌 学习目标

掌握 if / else if / else 分支，通过一个可运行的小程序理解核心用法。

知识点

  • if (条件) { ... } 条件为真才执行
  • else if 实现多分支，从上到下匹配
  • 比较运算符 == != < > <= >=，注意 = 是赋值
⚠️ 易错点
  • if (score = 90) 是赋值不是比较，会永远为真——用 ==
  • 分支条件要从高到低或从低到高排，避免逻辑漏洞
💡 课后练习

加一个「满分」等级（score == 100）。

main.c

#include <stdio.h>

/* 第 03 天：条件判断
 * 根据分数评定等级。
 */
const char* grade_of(int score) {
    if (score >= 90) return "优秀";
    else if (score >= 80) return "良好";
    else if (score >= 60) return "及格";
    else return "不及格";
}

int main(void) {
    int scores[] = {95, 82, 68, 45};
    int n = sizeof(scores) / sizeof(scores[0]);

    for (int i = 0; i < n; i++) {
        printf("分数 %d：%s\n", scores[i], grade_of(scores[i]));
    }
    return 0;
}

▶ 运行结果

分数 95：优秀
分数 82：良好
分数 68：及格
分数 45：不及格
