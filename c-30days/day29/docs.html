调试输出

条件编译和日志宏

📌 学习目标

掌握 条件编译和日志宏，通过一个可运行的小程序理解核心用法。

知识点

  • #ifdef DEBUG 控制调试代码的编译
  • 用宏封装调试输出，发布版一行 -DNDEBUG 全部关掉
  • __FILE__ __LINE__ __func__ 预定义宏定位代码位置
⚠️ 易错点
  • 调试宏的参数在发布版不执行，不要放有副作用的代码
  • printf 调试要带文件行号，否则不好定位
💡 课后练习

加一个 LOG_ERROR 宏，输出到 stderr。

main.c

#include <stdio.h>

/* 第 29 天：调试输出
 * 用宏实现可开关的日志。
 */

/* 定义 DEBUG 启用调试日志（编译时加 -DDEBUG）*/
#define DEBUG 1

/* 调试日志宏：带文件名和行号 */
#if DEBUG
    #define LOGD(fmt, ...) fprintf(stderr, "[DEBUG %s:%d] " fmt "\n", __FILE__, __LINE__, ##__VA_ARGS__)
#else
    #define LOGD(fmt, ...)  /* 发布版什么都不做 */
#endif

#define LOGI(fmt, ...) printf("[INFO] " fmt "\n", ##__VA_ARGS__)

int divide(int a, int b) {
    LOGD("调用 divide(%d, %d)", a, b);
    if (b == 0) {
        LOGD("除零，返回 0");
        return 0;
    }
    int result = a / b;
    LOGD("结果 = %d", result);
    return result;
}

int main(void) {
    LOGI("程序开始");
    LOGD("这是一条调试信息");

    printf("10 / 2 = %d\n", divide(10, 2));
    printf("10 / 0 = %d\n", divide(10, 0));

    LOGI("程序结束");
    return 0;
}

▶ 运行结果

[INFO] 程序开始
[DEBUG .../main.c:...] 调用 divide(10, 2)
[DEBUG .../main.c:...] 结果 = 5
10 / 2 = 5
[DEBUG .../main.c:...] 调用 divide(10, 0)
[DEBUG .../main.c:...] 除零，返回 0
10 / 0 = 0
[INFO] 程序结束
