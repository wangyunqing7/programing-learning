状态机

用枚举和 switch 写状态机

📌 学习目标

掌握 用枚举和 switch 写状态机，通过一个可运行的小程序理解核心用法。

知识点

  • 状态机 = 当前状态 + 输入 → 新状态
  • 用 enum 表示状态，switch 处理转移
  • 文字冒险、协议解析、UI 流程都是状态机
⚠️ 易错点
  • 状态转移表要覆盖所有状态×输入组合
  • 忘记 default 分支处理未知输入
💡 课后练习

加一个「战斗」状态，捡到武器才能战斗。

main.c

#include <stdio.h>

/* 第 25 天：状态机
 * 一个迷你文字冒险的状态机。
 */
typedef enum { S_START, S_FOREST, S_TREASURE, S_END } State;

const char* state_name(State s) {
    switch (s) {
        case S_START:    return "起点";
        case S_FOREST:   return "森林";
        case S_TREASURE: return "宝箱";
        case S_END:      return "结束";
    }
    return "?";
}

/* 模拟玩家的一系列输入 */
int main(void) {
    State state = S_START;
    char inputs[] = {'g', 'g', 't'};  /* g=go, t=take */
    int idx = 0, total = 3;

    while (state != S_END && idx < total) {
        char input = inputs[idx++];
        printf("[%s] 输入 %c -> ", state_name(state), input);

        switch (state) {
            case S_START:
                if (input == 'g') state = S_FOREST;
                break;
            case S_FOREST:
                if (input == 'g') state = S_TREASURE;
                break;
            case S_TREASURE:
                if (input == 't') { printf("拿到宝物！"); state = S_END; }
                break;
            default: break;
        }
        printf(" 现在 [%s]\n", state_name(state));
    }
    printf("冒险结束\n");
    return 0;
}

▶ 运行结果

[起点] 输入 g ->  现在 [森林]
[森林] 输入 g ->  现在 [宝箱]
[宝箱] 输入 t -> 拿到宝物！ 现在 [结束]
冒险结束

cd c-30days/day25
