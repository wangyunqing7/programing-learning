qsort

标准库快速排序和比较函数

📌 学习目标

掌握 标准库快速排序和比较函数，通过一个可运行的小程序理解核心用法。

知识点

  • qsort(数组, 个数, 元素大小, 比较函数)
  • 比较函数返回负数/0/正数表示小于/等于/大于
  • 比较函数收到的是 const void*，要强转成具体类型
⚠️ 易错点
  • 比较函数写错返回值，排序结果完全乱
  • strcmp 的返回值正好符合 qsort 的要求
💡 课后练习

用 qsort 对字符串数组按字典序排序。

main.c

#include <stdio.h>
#include <stdlib.h>

/* 第 15 天：qsort
 * 用标准库排序，学会写比较函数。
 */

/* 整数升序比较 */
int cmp_int(const void *a, const void *b) {
    int x = *(const int *)a;
    int y = *(const int *)b;
    return (x > y) - (x < y);  /* 避免溢出 */
}

int main(void) {
    int arr[] = {42, 7, 19, 88, 3, 56, 21};
    int n = sizeof(arr) / sizeof(arr[0]);

    printf("排序前：");
    for (int i = 0; i < n; i++) printf(" %d", arr[i]);
    printf("\n");

    qsort(arr, n, sizeof(int), cmp_int);

    printf("排序后：");
    for (int i = 0; i < n; i++) printf(" %d", arr[i]);
    printf("\n");
    return 0;
}

▶ 运行结果

排序前： 42 7 19 88 3 56 21
排序后： 3 7 19 21 42 56 88
