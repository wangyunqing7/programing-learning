Hello C

printf、main 函数和编译

📌 学习目标

掌握 printf、main 函数和编译，通过一个可运行的小程序理解核心用法。

知识点

  • main() 是 C 程序的入口，返回 int 给操作系统
  • printf 来自 <stdio.h>，用格式化串输出
  • \n 是换行，C 不会自动换行
⚠️ 易错点
  • 忘记 #include <stdio.h> 会导致 printf 未声明警告
  • 字符串必须用双引号，单引号是字符常量
💡 课后练习

改成打印你的名字和一句问候，编译运行看效果。

main.c

#include <stdio.h>

/* 第 01 天：Hello C
 * 掌握 main 入口和 printf 输出。
 * 编译：gcc main.c -o hello && ./hello
 */
int main(void) {
    printf("Hello, C learner\n");
    printf("今天开始 30 天 C 语言之旅\n");

    /* printf 可以一次输出多个值 */
    int day = 1;
    printf("现在是第 %d 天\n", day);

    /* 返回 0 表示程序正常结束 */
    return 0;
}

▶ 运行结果

Hello, C learner
今天开始 30 天 C 语言之旅
现在是第 1 天

编译运行

cd c-30days/day01
gcc -std=c11 -Wall main.c -o a && ./a

部分天数（如用到 math.h）需加 -lm 链接数学库。
