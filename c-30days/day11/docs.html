枚举

enum 和状态表示

📌 学习目标

掌握 enum 和状态表示，通过一个可运行的小程序理解核心用法。

知识点

  • enum { A, B, C }; 定义具名整数常量
  • 枚举让代码更可读：status = ERROR 比 status = 2 清楚
  • 枚举本质是 int，可以参与比较和 switch
⚠️ 易错点
  • 枚举值默认从 0 开始递增，可指定：enum { A=1, B, C };
  • 枚举不限制取值范围，赋任意 int 也不报错
💡 课后练习

用枚举表示交通灯，写一个循环模拟灯的变化。

main.c

#include <stdio.h>

/* 第 11 天：枚举
 * 用 enum 表示状态，提高可读性。
 */
typedef enum {
    STATE_OK = 0,
    STATE_WARNING = 1,
    STATE_ERROR = 2
} Status;

const char* status_name(Status s) {
    switch (s) {
        case STATE_OK:      return "正常";
        case STATE_WARNING: return "警告";
        case STATE_ERROR:   return "错误";
        default:            return "未知";
    }
}

Status check(int value) {
    if (value < 0)        return STATE_ERROR;
    else if (value > 100) return STATE_WARNING;
    else                  return STATE_OK;
}

int main(void) {
    int tests[] = {50, 120, -5};
    for (int i = 0; i < 3; i++) {
        Status s = check(tests[i]);
        printf("值 %3d -> 状态 %d (%s)\n", tests[i], s, status_name(s));
    }
    return 0;
}

▶ 运行结果

值  50 -> 状态 0 (正常)
值 120 -> 状态 1 (警告)
值  -5 -> 状态 2 (错误)
