数组统计

数组、遍历和统计

📌 学习目标

掌握 数组、遍历和统计，通过一个可运行的小程序理解核心用法。

知识点

  • int arr[5] 声明长度为 5 的数组，下标 0 到 4
  • sizeof(arr)/sizeof(arr[0]) 求数组元素个数
  • 数组名本身是首元素地址，不能直接赋值
⚠️ 易错点
  • 数组下标越界（访问 arr[5]）不会报错但行为未定义
  • 数组作为参数传递后会退化成指针，丢失长度信息
💡 课后练习

加一个计算中位数的函数。

main.c

#include <stdio.h>

/* 第 05 天：数组统计
 * 计算数组的和、平均、最大、最小。
 */
int main(void) {
    int nums[] = {12, 45, 7, 23, 56, 89, 34};
    int n = sizeof(nums) / sizeof(nums[0]);

    int sum = 0, max = nums[0], min = nums[0];
    for (int i = 0; i < n; i++) {
        sum += nums[i];
        if (nums[i] > max) max = nums[i];
        if (nums[i] < min) min = nums[i];
    }

    printf("数组：");
    for (int i = 0; i < n; i++) printf(" %d", nums[i]);
    printf("\n");
    printf("元素个数：%d\n", n);
    printf("总和：%d  平均：%.2f\n", sum, (double)sum / n);
    printf("最大：%d  最小：%d\n", max, min);
    return 0;
}

▶ 运行结果

数组： 12 45 7 23 56 89 34
元素个数：7
总和：266  平均：38.00
最大：89  最小：7
