循环求和

for / while 循环

📌 学习目标

掌握 for / while 循环，通过一个可运行的小程序理解核心用法。

知识点

  • for (初始化; 条件; 更新) 三段式循环
  • while (条件) 先判断后执行
  • break 跳出循环，continue 跳过本次
⚠️ 易错点
  • for (int i=0; i<=n; i++) 注意边界，< 还是 <=
  • 死循环：忘了更新循环变量
💡 课后练习

改成计算 1 到 n 的阶乘（注意溢出）。

main.c

#include <stdio.h>

/* 第 04 天：循环求和
 * 计算 1+2+...+n，演示 for 和 while。
 */
int main(void) {
    int n = 100;

    /* for 循环求和 */
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    printf("for: 1 到 %d 的和 = %d\n", n, sum);

    /* while 循环求和 */
    int i = 1, sum2 = 0;
    while (i <= n) {
        sum2 += i;
        i++;
    }
    printf("while: 1 到 %d 的和 = %d\n", n, sum2);

    /* break 提前退出 */
    for (int k = 1; k <= 10; k++) {
        if (k * k > 30) {
            printf("第一个平方超过 30 的是 %d\n", k);
            break;
        }
    }
    return 0;
}

▶ 运行结果

for: 1 到 100 的和 = 5050
while: 1 到 100 的和 = 5050
第一个平方超过 30 的是 6
