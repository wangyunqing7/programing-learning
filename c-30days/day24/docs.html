CSV 解析

文件读写 + 字符串解析

📌 学习目标

掌握 文件读写 + 字符串解析，通过一个可运行的小程序理解核心用法。

知识点

  • 用 fgets 逐行读文件
  • 用 strtok 按逗号分割每行
  • 用 sscanf 或 atoi 把字符串转数字
⚠️ 易错点
  • fgets 读入的行含换行符，要手动去掉
  • 最后一行可能没有换行符，处理要兼容
💡 课后练习

把解析结果写回一个新的 CSV，按总分排序。

main.c

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

/* 第 24 天：CSV 解析
 * 读取 CSV 文件并解析成结构体。
 */

/* 去掉行尾的换行符 */
void trim_newline(char *s) {
    size_t len = strlen(s);
    while (len > 0 && (s[len-1] == '\n' || s[len-1] == '\r')) {
        s[--len] = '\0';
    }
}

int main(void) {
    /* 先写一个 CSV 文件 */
    const char *path = "grades.csv";
    FILE *out = fopen(path, "w");
    fprintf(out, "Ada,95,88\nLinus,78,65\nGrace,55,72\n");
    fclose(out);

    /* 再读回来解析 */
    FILE *in = fopen(path, "r");
    if (!in) { printf("打开失败\n"); return 1; }

    char line[100];
    printf("%-10s %6s %6s %6s\n", "姓名", "数学", "英语", "合计");
    printf("------------------------------\n");

    while (fgets(line, sizeof(line), in)) {
        trim_newline(line);
        char *name = strtok(line, ",");
        char *m = strtok(NULL, ",");
        char *e = strtok(NULL, ",");
        if (name && m && e) {
            int math = atoi(m), eng = atoi(e);
            printf("%-10s %6d %6d %6d\n", name, math, eng, math + eng);
        }
    }
    fclose(in);
    remove(path);
    return 0;
}

▶ 运行结果

姓名        数学   英语    合计
------------------------------
Ada            95     88    183
Linus          78     65    143
Grace          55     72    127
