函数

函数定义、参数和返回值

📌 学习目标

掌握 函数定义、参数和返回值，通过一个可运行的小程序理解核心用法。

知识点

  • 返回类型 函数名(参数列表) { ... }
  • 参数是值传递，函数内改形参不影响实参
  • 函数声明（原型）放在调用前，定义可以放后面
⚠️ 易错点
  • 值传递：函数内改不了调用方的变量，要改得传指针
  • 函数没有 return 返回值时行为未定义（非 void 函数）
💡 课后练习

写一个判断素数的函数 is_prime。

main.c

#include <stdio.h>

/* 第 07 天：函数
 * 把功能拆成函数，演示值传递。
 */

/* 函数原型声明 */
int add(int a, int b);
int factorial(int n);

int main(void) {
    printf("add(3, 4) = %d\n", add(3, 4));
    printf("factorial(5) = %d\n", factorial(5));

    /* 值传递：传进去的是副本 */
    int x = 10;
    int dummy = add(x, 0);  /* x 不会被修改 */
    printf("x 调用后还是 %d\n", x);
    return 0;
}

int add(int a, int b) {
    return a + b;
}

int factorial(int n) {
    int result = 1;
    for (int i = 2; i <= n; i++) result *= i;
    return result;
}

▶ 运行结果

add(3, 4) = 7
factorial(5) = 120
x 调用后还是 10
