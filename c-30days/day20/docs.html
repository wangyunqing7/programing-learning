宏

#define 宏和预处理

📌 学习目标

掌握 #define 宏和预处理，通过一个可运行的小程序理解核心用法。

知识点

  • #define 名 值 定义常量或文本替换
  • 带参数的宏要给每个参数加括号：#define SQ(x) ((x)*(x))
  • 宏是文本替换，没有类型检查，复杂逻辑优先用函数
⚠️ 易错点
  • #define SQ(x) x*x 然后 SQ(1+2) 变成 1+2*1+2=5，不是 9
  • 宏参数有副作用时会执行多次：SQ(i++) 让 i 自增两次
💡 课后练习

写一个 MAX 宏和 MAX 函数，对比它们的差异。

main.c

#include <stdio.h>

/* 第 20 天：宏
 * 宏的定义与陷阱。
 */

#define PI 3.14159
#define MAX(a, b) ((a) > (b) ? (a) : (b))  /* 每个参数和整体都加括号 */
#define ARRAY_LEN(arr) (sizeof(arr) / sizeof((arr)[0]))

int main(void) {
    printf("PI = %f\n", PI);

    int x = 10, y = 20;
    printf("MAX(%d,%d) = %d\n", x, y, MAX(x, y));

    int arr[] = {1, 2, 3, 4, 5};
    printf("数组长度 = %zu\n", ARRAY_LEN(arr));

    /* 宏的陷阱：不加括号会出错 */
    printf("\n注意：MAX 宏已正确加括号，SQ(1+2) 类问题要用括号避免\n");
    return 0;
}

▶ 运行结果

PI = 3.141590
MAX(10,20) = 20
数组长度 = 5

注意：MAX 宏已正确加括号，SQ(1+2) 类问题要用括号避免
