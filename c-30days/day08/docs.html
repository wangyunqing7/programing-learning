结构体

struct 自定义数据类型

📌 学习目标

掌握 struct 自定义数据类型，通过一个可运行的小程序理解核心用法。

知识点

  • struct 名 { 字段; }; 定义结构，用 .成员 访问
  • 结构体可以整体赋值、作为函数参数和返回值
  • typedef 给结构体起别名，省去每次写 struct
⚠️ 易错点
  • 结构体定义后要有分号：struct P {int x;} ;
  • 用 -> 访问指针指向的结构体成员
💡 课后练习

给 Point 加一个计算两点距离的函数。

main.c

#include <stdio.h>
#include <math.h>

/* 第 08 天：结构体
 * 用 struct 把相关数据打包。
 */
typedef struct {
    double x;
    double y;
} Point;

double distance(Point a, Point b) {
    double dx = a.x - b.x;
    double dy = a.y - b.y;
    return sqrt(dx * dx + dy * dy);
}

void print_point(Point p) {
    printf("(%.1f, %.1f)", p.x, p.y);
}

int main(void) {
    Point p1 = {3.0, 4.0};
    Point p2 = {0.0, 0.0};

    printf("点1：");
    print_point(p1);
    printf("  点2：");
    print_point(p2);
    printf("\n距离：%.2f\n", distance(p1, p2));
    return 0;
}

▶ 运行结果

点1：(3.0, 4.0)  点2：(0.0, 0.0)
距离：5.00
