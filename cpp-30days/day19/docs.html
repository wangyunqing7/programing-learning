optional

std::optional 表示可能无值

📌 学习目标

掌握 std::optional 表示可能无值，通过一个可运行的小程序理解核心用法。

知识点

  • std::optional<T> 表示「有值或没有」
  • has_value()/value()/value_or() 安全访问
  • 替代用特殊值（-1、nullptr）表示失败的传统做法
⚠️ 易错点
  • 直接 value() 在无值时会抛异常，用 value_or 更安全
  • optional 有额外开销，别滥用
💡 课后练习

写一个在 map 中查找返回 optional 的函数。

main.cpp

#include <iostream>
#include <optional>
#include <vector>
#include <string>

// 第 19 天：optional
std::optional<int> find_first_even(const std::vector<int>& v) {
    for (auto n : v) {
        if (n % 2 == 0) return n;  // 找到返回值
    }
    return std::nullopt;  // 找不到返回空
}

int main() {
    std::vector<int> a = {1, 3, 5, 8, 9};
    std::vector<int> b = {1, 3, 5};

    if (auto r = find_first_even(a)) {
        std::cout << "第一个偶数：" << *r << std::endl;
    }
    auto r2 = find_first_even(b);
    std::cout << "无偶数时：" << r2.value_or(-1) << std::endl;

    // has_value 检查
    std::cout << "a 有偶数？" << (find_first_even(a).has_value() ? "是" : "否") << std::endl;
    return 0;
}

▶ 运行结果

第一个偶数：8
无偶数时：-1
a 有偶数？是
