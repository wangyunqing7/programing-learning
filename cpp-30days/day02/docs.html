数字计算

变量、auto 和表达式

📌 学习目标

掌握 变量、auto 和表达式，通过一个可运行的小程序理解核心用法。

知识点

  • auto 让编译器推断类型，简化书写
  • const 定义常量，编译期或运行期
  • constexpr 定义编译期常量表达式
⚠️ 易错点
  • auto 推断引用和 const 要显式写 auto&
  • 整数除法仍会截断
💡 课后练习

加一个用 constexpr 计算数组大小的例子。

main.cpp

#include <iostream>

// 第 02 天：数字计算
// 掌握 auto 和 const。
int main() {
    constexpr double PI = 3.14159265;
    auto radius = 5.0;            // auto 推断为 double
    auto area = PI * radius * radius;

    std::cout << "半径 " << radius << " 的圆面积 = " << area << std::endl;

    // 整数 vs 浮点
    auto a = 7, b = 2;
    std::cout << "7/2 = " << a / b << " (整数截断)" << std::endl;
    std::cout << "7.0/2 = " << 7.0 / b << std::endl;
    std::cout << "7%2 = " << a % b << std::endl;
    return 0;
}

▶ 运行结果

半径 5 的圆面积 = 78.5398
7/2 = 3 (整数截断)
7.0/2 = 3.5
7%2 = 1
