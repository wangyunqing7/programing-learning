枚举

enum class 强类型枚举

📌 学习目标

掌握 enum class 强类型枚举，通过一个可运行的小程序理解核心用法。

知识点

  • enum class 不会隐式转 int，更安全
  • 作用域：Color::Red 必须带枚举名
  • 可指定底层类型：enum class E : int
⚠️ 易错点
  • 老式 enum 会污染命名空间且隐式转 int
  • enum class 不能直接 cout，要转 int
💡 课后练习

用 enum class 表示订单状态并写状态转移。

main.cpp

#include <iostream>

// 第 11 天：枚举
enum class Color { Red, Green, Blue };
enum class Status { Pending = 0, Active = 1, Done = 2 };

std::string status_name(Status s) {
    switch (s) {
        case Status::Pending: return "待处理";
        case Status::Active:  return "进行中";
        case Status::Done:    return "已完成";
    }
    return "?";
}

int main() {
    Color c = Color::Green;
    Status s = Status::Active;

    // enum class 不会隐式转 int
    std::cout << "状态：" << status_name(s) << std::endl;
    std::cout << "颜色编号：" << static_cast<int>(c) << std::endl;

    // 安全比较
    if (s == Status::Active) {
        std::cout << "任务正在进行" << std::endl;
    }
    return 0;
}

▶ 运行结果

状态：进行中
颜色编号：1
任务正在进行

编译运行

cd cpp-30days/day11

需要 C++17 或更高（用到结构化绑定、filesystem 等）。
