字符串处理

std::string 的丰富方法

📌 学习目标

掌握 std::string 的丰富方法，通过一个可运行的小程序理解核心用法。

知识点

  • std::string 自动管理内存，比 C 字符串安全
  • substr/find/append/replace 常用方法
  • to_string/stoi 字符串与数字互转
⚠️ 易错点
  • find 找不到返回 std::string::npos，不是 -1
  • 频繁拼接用 += 或 reserve 提高性能
💡 课后练习

实现一个简单的字符串分割函数（按空格分词）。

main.cpp

#include <iostream>
#include <string>

// 第 06 天：字符串处理
int main() {
    std::string s = "Hello, C++ learner";

    std::cout << "长度：" << s.length() << std::endl;
    std::cout << "子串(0,5)：" << s.substr(0, 5) << std::endl;

    // find 查找
    auto pos = s.find("C++");
    if (pos != std::string::npos) {
        std::cout << "找到 C++ 在位置 " << pos << std::endl;
    }

    // 拼接和替换
    s += "!";
    s.replace(7, 3, "modern C++");
    std::cout << "处理后：" << s << std::endl;

    // 数字与字符串互转
    int n = 42;
    std::string num_str = std::to_string(n);
    int parsed = std::stoi("123abc");  // 解析到非数字为止
    std::cout << num_str << " / " << parsed << std::endl;
    return 0;
}

▶ 运行结果

长度：17
子串(0,5)：Hello
找到 C++ 在位置 7
处理后：Hello, modern C++ learner!
42 / 123

编译运行

cd cpp-30days/day06
g++ -std=c++17 -Wall main.cpp -o a && ./a

需要 C++17 或更高（用到结构化绑定、filesystem 等）。
