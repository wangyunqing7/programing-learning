构造函数

构造、初始化列表和析构

📌 学习目标

掌握 构造、初始化列表和析构，通过一个可运行的小程序理解核心用法。

知识点

  • 类(参数) : 成员(值) {} 初始化列表更高效
  • 析构函数 ~类() 对象销毁时调用，释放资源
  • 拷贝构造、移动构造管理对象复制
⚠️ 易错点
  • 成员初始化顺序按声明顺序，不按初始化列表顺序
  • 有动态资源要遵循三五法则
💡 课后练习

加一个拷贝构造函数观察拷贝发生。

main.cpp

#include <iostream>
#include <string>

// 第 10 天：构造函数
class Book {
    std::string title;
    int pages;
public:
    // 初始化列表
    Book(std::string t, int p) : title(std::move(t)), pages(p) {
        std::cout << "构造：" << title << std::endl;
    }
    ~Book() {
        std::cout << "析构：" << title << std::endl;
    }
    void show() const { std::cout << "《" << title << "》" << pages << "页" << std::endl; }
};

int main() {
    Book b1("流畅的C++", 300);
    b1.show();

    {
        Book b2("临时书", 50);  // 出作用域自动析构
        b2.show();
    }  // b2 在这里析构
    std::cout << "回到 main" << std::endl;
    return 0;
}

▶ 运行结果

构造：流畅的C++
《流畅的C++》300页
构造：临时书
《临时书》50页
析构：临时书
回到 main
析构：流畅的C++

cd cpp-30days/day10
