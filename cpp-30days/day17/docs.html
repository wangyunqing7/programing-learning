模板函数

template 泛型编程

📌 学习目标

掌握 template 泛型编程，通过一个可运行的小程序理解核心用法。

知识点

  • template<typename T> 定义泛型函数或类
  • 编译器根据调用自动实例化不同类型版本
  • STL 容器和算法大量使用模板
⚠️ 易错点
  • 模板定义通常放头文件（编译器需要完整定义）
  • 模板错误信息冗长难读
💡 课后练习

写一个泛型的 max_of 模板。

main.cpp

#include <iostream>
#include <vector>
#include <string>

// 第 17 天：模板函数
template <typename T>
T my_max(T a, T b) {
    return a > b ? a : b;
}

template <typename T>
T sum(const std::vector<T>& v) {
    T total = T();
    for (const auto& x : v) total += x;
    return total;
}

int main() {
    std::cout << "max(3, 7) = " << my_max(3, 7) << std::endl;
    std::cout << "max(2.5, 1.8) = " << my_max(2.5, 1.8) << std::endl;
    std::cout << R"(max("apple","banana") = )" << my_max(std::string("apple"), std::string("banana")) << std::endl;

    std::vector<int> nums = {1, 2, 3, 4, 5};
    std::cout << "sum = " << sum(nums) << std::endl;
    return 0;
}

▶ 运行结果

max(3, 7) = 7
max(2.5, 1.8) = 2.5
max("apple","banana") = banana
sum = 15

编译运行

cd cpp-30days/day17
g++ -std=c++17 -Wall main.cpp -o a && ./a

需要 C++17 或更高（用到结构化绑定、filesystem 等）。
