map 统计

std::map 键值容器

📌 学习目标

掌握 std::map 键值容器，通过一个可运行的小程序理解核心用法。

知识点

  • std::map 按键有序存储，unordered_map 哈希无序更快
  • [key] 访问，不存在会创建默认值
  • 遍历得到 pair，.first 键 .second 值
⚠️ 易错点
  • [key] 会插入空值，只想查询用 .find 或 .at
  • 自定义类型做 key 要提供比较函数
💡 课后练习

统计一段文本的字符频率。

main.cpp

#include <iostream>
#include <map>
#include <vector>
#include <string>

// 第 12 天：map 统计
int main() {
    std::vector<std::string> words = {"apple","banana","apple","cherry","banana","apple"};

    // 统计词频
    std::map<std::string, int> freq;
    for (const auto& w : words) {
        freq[w]++;  // 不存在则创建为0再+1
    }

    std::cout << "词频统计：" << std::endl;
    for (const auto& [word, count] : freq) {  // 结构化绑定 (C++17)
        std::cout << "  " << word << " : " << count << std::endl;
    }

    // 安全查询
    auto it = freq.find("grape");
    std::cout << "grape 存在？" << (it != freq.end() ? "是" : "否") << std::endl;
    return 0;
}

▶ 运行结果

词频统计：
  apple : 3
  banana : 2
  cherry : 1
grape 存在？否
