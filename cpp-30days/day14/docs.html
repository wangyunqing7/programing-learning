异常处理

throw / try / catch

📌 学习目标

掌握 throw / try / catch，通过一个可运行的小程序理解核心用法。

知识点

  • throw 抛出异常对象，try/catch 捕获
  • 标准异常继承 std::exception，.what() 取信息
  • 异常让错误处理与正常逻辑分离
⚠️ 易错点
  • catch 顺序从具体到一般，catch(...) 兜底
  • 析构函数里别抛异常
💡 课后练习

自定义一个继承 std::exception 的异常类。

main.cpp

#include <iostream>
#include <stdexcept>
#include <string>

// 第 14 天：异常处理
double safe_divide(int a, int b) {
    if (b == 0) throw std::invalid_argument("除数为0");
    if (a % b != 0) throw std::runtime_error("不能整除");
    return static_cast<double>(a) / b;
}

int main() {
    std::pair<int,int> cases[] = {{10,2},{10,0},{10,3}};
    for (auto [a, b] : cases) {
        try {
            std::cout << a << "/" << b << " = " << safe_divide(a, b) << std::endl;
        } catch (const std::exception& e) {
            std::cout << a << "/" << b << " 失败：" << e.what() << std::endl;
        }
    }
    return 0;
}

▶ 运行结果

10/2 = 5
10/0 失败：除数为0
10/3 失败：不能整除

编译运行

cd cpp-30days/day14
g++ -std=c++17 -Wall main.cpp -o a && ./a

需要 C++17 或更高（用到结构化绑定、filesystem 等）。
