文件输出

fstream 文件读写

📌 学习目标

掌握 fstream 文件读写，通过一个可运行的小程序理解核心用法。

知识点

  • std::ofstream 写文件，std::ifstream 读
  • std::fstream 可读可写
  • RAII：对象析构自动关闭文件
⚠️ 易错点
  • 打开文件要检查 is_open()
  • 默认覆盖，追加要 std::ios::app
💡 课后练习

实现一个追加模式的日志写入。

main.cpp

#include <iostream>
#include <fstream>
#include <string>

// 第 13 天：文件输出
int main() {
    const std::string path = "output.txt";

    // 写文件（RAII 自动关闭）
    {
        std::ofstream out(path);
        if (!out.is_open()) { std::cerr << "打开失败" << std::endl; return 1; }
        out << "姓名 数学 英语\n";
        out << "Ada 95 88\n";
        out << "Linus 78 65\n";
    }  // out 析构自动关闭

    // 读文件
    std::ifstream in(path);
    std::cout << "--- 文件内容 ---" << std::endl;
    std::string line;
    while (std::getline(in, line)) {
        std::cout << line << std::endl;
    }

    std::remove(path.c_str());
    return 0;
}

▶ 运行结果

--- 文件内容 ---
姓名 数学 英语
Ada 95 88
Linus 78 65
