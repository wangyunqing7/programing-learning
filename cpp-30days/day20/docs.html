chrono

时间处理和计时

📌 学习目标

掌握 时间处理和计时，通过一个可运行的小程序理解核心用法。

知识点

  • std::chrono 类型安全的时钟和时长
  • steady_clock 单调递增，适合测耗时
  • system_clock::now() 获取当前时间
⚠️ 易错点
  • 不要用 time()/clock()，chrono 更现代安全
  • 不同时钟精度不同
💡 课后练习

测量一个循环一百万次的耗时。

main.cpp

#include <iostream>
#include <chrono>
#include <thread>

// 第 20 天：chrono
int main() {
    auto start = std::chrono::steady_clock::now();

    // 模拟工作：累加
    long long sum = 0;
    for (int i = 0; i < 1000000; i++) sum += i;

    auto end = std::chrono::steady_clock::now();
    auto ms = std::chrono::duration_cast<std::chrono::microseconds>(end - start);
    std::cout << "累加 100 万次耗时：" << ms.count() << " 微秒" << std::endl;
    std::cout << "sum = " << sum << std::endl;

    // 休眠（演示）
    auto t0 = std::chrono::steady_clock::now();
    std::this_thread::sleep_for(std::chrono::milliseconds(10));
    auto t1 = std::chrono::steady_clock::now();
    std::cout << "休眠约 " << std::chrono::duration_cast<std::chrono::milliseconds>(t1-t0).count() << " 毫秒" << std::endl;
    return 0;
}
