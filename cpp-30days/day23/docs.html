sstream

字符串流与格式化

📌 学习目标

掌握 字符串流与格式化，通过一个可运行的小程序理解核心用法。

知识点

  • std::ostringstream 往字符串写入（拼字符串）
  • std::istringstream 从字符串读出（解析）
  • 比 sprintf/sscanf 更类型安全
⚠️ 易错点
  • sstream 性能略低，热点循环慎用
  • getline 可按分隔符切分
💡 课后练习

用 istringstream 解析一行 CSV 数据。

main.cpp

#include <iostream>
#include <sstream>
#include <string>

// 第 23 天：sstream
int main() {
    // ostringstream：拼接字符串
    std::ostringstream oss;
    oss << "学生：" << "Ada" << " 分数：" << 95;
    std::string result = oss.str();
    std::cout << result << std::endl;

    // istringstream：解析字符串
    std::string data = "Ada 95 88";
    std::istringstream iss(data);
    std::string name;
    int math, english;
    iss >> name >> math >> english;
    std::cout << name << " 总分 " << math + english << std::endl;

    // 字符串转数字
    std::istringstream num("42");
    int n;
    num >> n;
    std::cout << "解析的数字：" << n << std::endl;
    return 0;
}

▶ 运行结果

学生：Ada 分数：95
Ada 总分 183
解析的数字：42
