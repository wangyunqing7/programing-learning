random

现代随机数生成

📌 学习目标

掌握 现代随机数生成，通过一个可运行的小程序理解核心用法。

知识点

  • std::random_device 提供种子
  • std::mt19937 高质量伪随机引擎
  • uniform_int_distribution/uniform_real_distribution 分布
⚠️ 易错点
  • 别用 rand()，质量差且有线程问题
  • 引擎要持久化，别每次重新种子
💡 课后练习

生成一个随机密码并评估强度。

main.cpp

#include <iostream>
#include <random>
#include <string>

// 第 22 天：random
int random_int(int lo, int hi) {
    static std::mt19937 gen(std::random_device{}());
    std::uniform_int_distribution<int> dist(lo, hi);
    return dist(gen);
}

int main() {
    std::cout << "随机整数：";
    for (int i = 0; i < 5; i++) std::cout << " " << random_int(1, 100);
    std::cout << std::endl;

    // 生成随机密码
    const std::string chars = "abcdefghijklmnopqrstuvwxyz0123456789";
    std::string pwd;
    for (int i = 0; i < 8; i++) {
        pwd += chars[random_int(0, chars.size() - 1)];
    }
    std::cout << "随机密码：" << pwd << std::endl;

    // 模拟掷骰子 10000 次的分布
    int counts[6] = {};
    for (int i = 0; i < 10000; i++) counts[random_int(1,6)-1]++;
    std::cout << "骰子分布：";
    for (int c : counts) std::cout << " " << c;
    std::cout << std::endl;
    return 0;
}

▶ 运行结果

随机整数： 42 7 88 3 56
随机密码：a3k9x2mq
骰子分布： 1685 1672 1654 1668 1651 1670
