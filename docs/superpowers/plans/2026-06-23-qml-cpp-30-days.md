# QML + C++ 30-Day Tutorial Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete 30-day QML + C++ tutorial with runnable Qt projects and daily HTML explanations.

**Architecture:** A Python generator writes a consistent tutorial tree, daily Qt projects, shared documentation styling, and per-day build scripts. Each day remains an independent CMake project so the learner can open or build any day without depending on previous days.

**Tech Stack:** Qt 6.11.1, Qt Quick, C++, CMake, Ninja, MinGW, HTML/CSS, Python generation script.
