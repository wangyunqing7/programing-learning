学习建议

  • 先运行项目，观察输出或页面效果。
  • 再阅读源码，把变量、函数、数据结构和控制流画出来。
  • 最后完成 README 里的今日练习，哪怕只改一小处也要重新运行。

常见错误

  • 路径里有空格时，请在 PowerShell 中用引号包住路径。
  • 如果脚本被执行策略拦截，可以使用 powershell -ExecutionPolicy Bypass -File .\run.ps1。
  • 修改代码后要重新运行或重新编译，不要只看旧输出。
