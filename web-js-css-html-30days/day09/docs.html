颜色选择器

input color 和 CSS 变量

📌 学习目标

做一个 颜色选择器，掌握 input color 和 CSS 变量。

知识点

  • input type color 原生取色器
  • setProperty 改 CSS 变量
  • 实时预览
⚠️ 易错点
  • CSS 变量一处定义处处用
  • 颜色值是 #rrggbb
💡 课后练习

加随机配色按钮。

▶ 在浏览器打开成品 →

index.html（节选）

<!doctype html><html lang="zh-CN"><head><meta charset="utf-8"><title>Day 09 · 颜色选择器</title><style>:root{--m:#0f5cc0}body{font-family:system-ui;background:#f0f4f8;margin:0;padding:40px;text-align:center}.pv{background:var(--m);color:#fff;padding:60px;border-radius:16px;margin:20px auto;max-width:400px;transition:.3s}input{width:80px;height:40px;border:none;cursor:pointer}</style></head><body><h2>选择主题色</h2><input type="color" id="pk" value="#0f5cc0"><div class="pv"><h1>实时预览</h1><p>颜色立即应用</p></div><script>document.getElementById("pk").addEventListener("input",e=>document.documentElement.style.setProperty("--m",e.target.value))</script></body></html>
