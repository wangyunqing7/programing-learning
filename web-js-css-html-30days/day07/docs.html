进度条

CSS 宽度和 JS 状态

📌 学习目标

做一个 进度条，掌握 CSS 宽度和 JS 状态。

知识点

  • div 宽度百分比控制进度
  • transition 让变化平滑
  • setInterval 模拟任务
⚠️ 易错点
  • 宽度要 transition
  • setInterval 完要 clear
💡 课后练习
