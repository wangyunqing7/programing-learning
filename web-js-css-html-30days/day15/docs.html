键盘快捷键

keydown 事件

📌 学习目标

做一个 键盘快捷键，掌握 keydown 事件。

知识点

  • keydown 监听键盘
  • e.key 判断按键
  • 组合键 ctrlKey/altKey
⚠️ 易错点
  • 注意按键大小写
  • 某些要 preventDefault
