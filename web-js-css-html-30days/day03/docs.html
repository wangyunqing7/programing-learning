主题切换

CSS class 和状态切换

📌 学习目标

做一个 主题切换，掌握 CSS class 和状态切换。

知识点

  • classList.toggle 切换类
  • CSS 用 .dark 选择器定义暗色
  • localStorage 保存偏好
⚠️ 易错点
  • CSS 变量让切换更简洁
  • 刷新会丢失，要存 localStorage
💡 课后练习

加跟随系统主题。

▶ 在浏览器打开成品 →

index.html（节选）

<!doctype html><html lang="zh-CN"><head><meta charset="utf-8"><title>Day 03 · 主题切换</title><style>:root{--bg:#f0f4f8;--card:#fff;--text:#182033}.dark{--bg:#182033;--card:#273449;--text:#e2e8f0}body{font-family:system-ui;background:var(--bg);color:var(--text);margin:0;min-height:100vh;display:flex;justify-content:center;align-items:center;transition:.3s}.card{background:var(--card);padding:40px;border-radius:16px;text-align:center;transition:.3s}button{padding:10px 24px;border:none;border-radius:8px;background:#0f5cc0;color:#fff;cursor:pointer}</style></head><body><div class="card"><h2>切换主题</h2><p>当前主题保存到本地</p><button id="t">切换亮/暗</button></div><script>if(localStorage.getItem("theme")==="dark")document.body.classList.add("dark");document.getElementById("t").addEventListener("click",()=>{document.body.classList.toggle("dark");localStorage.setItem("theme",document.body.classList.contains("dark")?"dark":"light")})</script></body></html>
