Markdown 预览

文本转换

📌 学习目标

做一个 Markdown 预览，掌握 文本转换。

知识点

  • 实时把 Markdown 转 HTML
  • 简单实现标题加粗列表
  • 输入即预览
⚠️ 易错点
  • 转 HTML 防 XSS 转义
  • 正则顺序有讲究
