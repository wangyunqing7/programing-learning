计数器

按钮事件和 DOM 更新

📌 学习目标

做一个 计数器，掌握 按钮事件和 DOM 更新。

知识点

  • addEventListener 绑定点击
  • 事件处理函数修改状态再更新 DOM
  • textContent 改文本
⚠️ 易错点
  • 变量要先声明
  • 直接改 DOM 比操作 innerHTML 更安全
💡 课后练习

加一个重置按钮。
