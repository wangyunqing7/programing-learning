模态框

显示隐藏交互

📌 学习目标

做一个 模态框，掌握 显示隐藏交互。

知识点

  • 默认 display none 点击切换
  • 遮罩层点击关闭
  • opacity 过渡淡入
⚠️ 易错点
  • 遮罩覆盖全屏置顶层
  • 点内容不关闭要 stopPropagation
💡 课后练习

加 ESC 关闭。

▶ 在浏览器打开成品 →
