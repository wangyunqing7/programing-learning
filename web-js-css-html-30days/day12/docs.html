标签页

tab 状态切换

📌 学习目标

做一个 标签页，掌握 tab 状态切换。

知识点

  • 共用激活状态
  • 点击切换 active 并显示对应内容
  • 默认显示第一个
⚠️ 易错点
  • tab 和内容要对应
  • 事件用索引
💡 课后练习

加滑动切换。
