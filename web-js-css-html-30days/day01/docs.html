个人名片

HTML 结构、CSS 基础、JS 改文本

📌 学习目标

做一个 个人名片，掌握 HTML 结构、CSS 基础、JS 改文本。

知识点

  • HTML 用标签描述结构：div/p/h1
  • CSS 选择器 .class、#id 控制样式
  • JS querySelector 选取元素改 textContent
⚠️ 易错点
  • JS 要放 body 末尾，否则取不到元素
  • 类名用点，id 用井号
💡 课后练习

加一个技能列表，按钮切换显示。

▶ 在浏览器打开成品 →

index.html（节选）

<!doctype html>
<html lang="zh-CN">
<head>
<meta charset="utf-8">
<meta name="viewport" content="width=device-width,initial-scale=1">
<title>Day 01 · 个人名片</title>
<style>
  body{font-family:system-ui;background:#f0f4f8;margin:0;padding:40px;display:flex;justify-content:center}
  .card{background:#fff;border-radius:16px;padding:32px;width:320px;box-shadow:0 10px 30px rgba(0,0,0,.1);text-align:center}
  .avatar{width:96px;height:96px;border-radius:50%;background:linear-gradient(135deg,#0f5cc0,#0d9488);margin:0 auto 16px;display:flex;align-items:center;justify-content:center;color:#fff;font-size:36px;font-weight:bold}
  h1{margin:0 0 4px;color:#182033}
  .title{color:#64748b;margin:0 0 16px}
  .bio{color:#475569;line-height:1.6;font-size:14px}
  button{margin-top:16px;padding:8px 20px;border:none;border-radius:8px;background:#0f5cc0;color:#fff;cursor:pointer;font-size:14px}
  button:hover{background:#0b4a99}
</style>
</head>
<body>
<div class="card">
  <div class="avatar" id="avatar">A</div>
  ...（完整见 index.html）
