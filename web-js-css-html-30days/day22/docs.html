CSS 动画卡片

transition 和 animation

📌 学习目标

做一个 CSS 动画卡片，掌握 transition 和 animation。

知识点

  • transition 状态过渡
  • @keyframes 关键帧
  • hover/transform 悬停
⚠️ 易错点
  • transform 性能好
  • 动画属性要明确
💡 课后练习

加 3D 翻转。
