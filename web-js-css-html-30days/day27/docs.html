天气卡片

组件化渲染

📌 学习目标

做一个 天气卡片，掌握 组件化渲染。

知识点

  • 数据数组渲染多个卡片
  • 卡片可复用结构
  • 模拟天气数据
⚠️ 易错点
  • 组件化可复用
  • 数据变只更新一处
