Fetch 模拟数据

Promise 和异步

📌 学习目标

做一个 Fetch 模拟数据，掌握 Promise 和异步。

知识点

  • fetch 返回 Promise
  • async/await 像同步
  • then/catch 处理
⚠️ 易错点
  • fetch 不把 404 当异常
  • await 要在 async 函数
