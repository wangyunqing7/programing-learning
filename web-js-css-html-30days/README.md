# 前端 30 天学习项目（HTML / CSS / JavaScript）

每天一个可交互的网页小项目，从个人名片到计算器、番茄钟、看板和作品集。

## 怎么学

每个 day 目录里有两个文件：

- `index.html` —— 当天的**成品页面**（HTML + CSS + JS 单文件），双击用浏览器打开即可看到效果并交互。
- `docs.html` —— 当天的**讲义**（左右分栏：学习目标、知识点、易错点、练习，右侧代码节选）。

从 `index.html`（本目录）进入可以看到全部 30 天的卡片导航，按周分组。

## 课程结构

| 周 | 主题 | 天数 |
|----|------|------|
| 第 1 周 | 基础交互（名片、计数器、主题、待办、表单、画廊、进度条）| day01–07 |
| 第 2 周 | 常用组件（计算器、颜色、过滤、模态框、标签页、折叠、备忘录）| day08–14 |
| 第 3 周 | 进阶交互（快捷键、拖放、Canvas、Fetch、响应式、表格、搜索）| day15–21 |
| 第 4 周 | 动效与综合（动画、测验、番茄钟、Markdown、购物、天气、看板、作品集）| day22–30 |

每天 `docs.html` 顶部有进度条，底部有前/后导航。

## 特点

- 每个成品都是**单文件**（内联 CSS + JS），打开即用，无需服务器或构建工具。
- 纯原生 HTML/CSS/JS，不依赖任何框架，适合打基础。
- 讲义页和成品页分离：docs.html 学原理，index.html 看效果。
