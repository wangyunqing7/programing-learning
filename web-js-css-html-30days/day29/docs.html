项目看板

多列数据渲染

📌 学习目标

做一个 项目看板，掌握 多列数据渲染。

知识点

  • 看板分待办/进行中/完成
  • 任务在列间移动
  • 每列独立渲染
⚠️ 易错点
  • 移动要更新数据再渲染
  • 列数据分离
💡 课后练习

加拖拽移动。
