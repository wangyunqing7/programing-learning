可访问性表单

label、aria、焦点

📌 学习目标

做一个 可访问性表单，掌握 label、aria、焦点。

知识点

  • label for 关联输入
  • aria-label 加描述
  • :focus 焦点样式
⚠️ 易错点
  • 表单都要有 label
  • 按钮文字要明确
💡 课后练习

加 tabindex 焦点顺序。
