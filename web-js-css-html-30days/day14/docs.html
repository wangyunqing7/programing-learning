本地备忘录

localStorage 基础

📌 学习目标

做一个 本地备忘录，掌握 localStorage 基础。

知识点

  • setItem/getItem 存取
  • JSON.stringify/parse 存对象
  • 加载时读，修改时存
⚠️ 易错点
  • 只能存字符串
  • 容量约 5MB
💡 课后练习

加删除功能。
