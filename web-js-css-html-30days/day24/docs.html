番茄钟

计时器

📌 学习目标

做一个 番茄钟，掌握 计时器。

知识点

  • setInterval 每秒更新
  • 分:秒格式化
  • 工作和休息交替
⚠️ 易错点
  • setInterval 完要 clear
  • 切换模式重置
