标签过滤

filter 和数据属性

📌 学习目标

做一个 标签过滤，掌握 filter 和数据属性。

知识点

  • data-* 给元素打标签
  • filter 按条件筛选
  • 点击标签重新渲染
⚠️ 易错点
  • 全部标签要单独处理
  • dataset 读取 data 属性
