CSV 成绩表

csv 读写、聚合统计、排序

📌 学习目标

  • 用 csv.DictWriter/DictReader 读写 CSV
  • 理解为什么要 newline=''
  • 读 CSV 时做类型转换（字符串→int）
  • 用 sorted(key=...) 多条件排序

知识点

DictWriter：按字段名写，writeheader() 写表头，writerows() 批量写。

newline=''：Windows 上 csv 模块要这么开文件，否则会产生空行。

类型转换：CSV 读出来全是字符串，分数要 int(r['math']) 转换才能比较/相加。

排序 key：key=lambda r: -(r['math']+...) 用负号实现降序。

⚠️ 易错点
  • Windows 下不写 newline='' 会有空行。
  • 忘做类型转换，字符串比较会按字典序（'9' > '80' 出错）。
💡 课后练习
  • 按科目分别生成单独的 CSV。
  • 导出成带格式的 Markdown 表格。

import csv

def write_csv(path, rows):
    with path.open("w", newline="", encoding="utf-8") as f:
        w = csv.DictWriter(f, fieldnames=["name","math","english","python"])
        w.writeheader()
