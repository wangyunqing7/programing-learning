CLI 菜单模拟

函数分发、会话循环、命令表

📌 学习目标

  • 用「命令名→方法名」字典做分发
  • 用 getattr 按名字调用方法
  • 用 split(maxsplit=1) 拆命令和参数
  • 设计可循环的会话

知识点

分发表：{'add': 'cmd_add', ...} 把命令字符串映射到方法名，getattr(obj, name)(arg) 动态调用，避免写一长串 if/elif。

maxsplit：line.split(maxsplit=1) 只切第一刀，add 学习 Python 会切成 ['add', '学习 Python']，任务名里的空格不被拆散。

会话状态：把 tasks、running 等放进 Session 对象，命令方法修改它。

⚠️ 易错点
  • getattr 找不到方法会抛 AttributeError，要先判断 if name in COMMANDS。
  • 用真实 input() 时要有退出命令，否则死循环。
💡 课后练习
  • 把 run_script 改成真实 input() 循环。
  • 加 save/load 命令持久化任务。

main.py（核心）

COMMANDS = {"add": "cmd_add", "list": "cmd_list",
           "done": "cmd_done", "exit": "cmd_exit"}

def run_script(session, lines):
    for line in lines:
        parts = line.split(maxsplit=1)
        cmd, arg = parts[0], (parts[1] if len(parts) > 1 else "")
        method = COMMANDS.get(cmd)
        if method is None:
            print(f"未知命令：{cmd}"); continue
        getattr(session, method)(arg)

▶ 运行结果

> help
  命令：add <任务> | list | done <序号> | help | exit
> add 学完 day16
  已添加：学完 day16
> list
  1. 学完 day16
> bogus
  未知命令：bogus
