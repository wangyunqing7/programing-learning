温度换算器

数字运算、表达式、格式化输出

📌 学习目标

  • 掌握算术运算符和运算优先级
  • 用 f"{x:.2f}" 控制小数位数与对齐
  • 学会 raise 抛出异常、try/except 捕获
  • 用「先归一再转换」简化多种单位互转

知识点

归一化思路：摄氏 C、华氏 F、开尔文 K 三者两两组合要写 6 个函数；但如果都以摄氏为「中转站」，只需 C→F、C→K 和它们的逆，逻辑更清晰。

格式化：{value:>7.2f} 表示右对齐、总宽 7、保留 2 位小数，适合做表格对齐。

异常：用 raise ValueError(...) 表示业务错误，调用方用 try/except 决定怎么处理（打印提示而不是崩溃）。

⚠️ 易错点
  • 9 / 5 和 5 / 9 别写反，否则结果差很多。
  • 物理上存在绝对零度（-273.15°C），低于它要报错而不是给出无意义数字。
💡 课后练习
  • 加一个兰氏度（Rankine，R）单位，R = (C + 273.15) × 1.8。
  • 把 samples 改成从用户输入读取。

main.py（核心）

ABSOLUTE_ZERO_C = -273.15

def c_to_f(c): return c * 9 / 5 + 32
def f_to_c(f): return (f - 32) * 5 / 9
def c_to_k(c): return c + 273.15
def k_to_c(k): return k - 273.15

def convert(value, unit):
    unit = unit.upper()
    if unit == "C":   c = value
    elif unit == "F": c = f_to_c(value)
    elif unit == "K": c = k_to_c(value)
    else: raise ValueError(f"不支持：{unit}")
    if c < ABSOLUTE_ZERO_C:
        raise ValueError("低于绝对零度")
    return {"C": c, "F": c_to_f(c), "K": c_to_k(c)}
