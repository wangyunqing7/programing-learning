JSON 任务数据

json 序列化、嵌套结构、持久化

📌 学习目标

  • 用 json.dumps/loads 序列化和反序列化
  • 设计嵌套的 dict 结构（项目→任务）
  • 用 @classmethod 做反序列化工厂
  • 处理 ensure_ascii 中文问题

知识点

序列化：json.dumps(obj) 把 Python 对象转成 JSON 字符串，loads(s) 反过来。

中文：默认 ensure_ascii=True 会把中文转成 \uXXXX，设成 False 才保留中文。

classmethod：@classmethod 的第一个参数是类本身，常用来写「从字典/JSON 构造实例」的工厂方法。

⚠️ 易错点
  • ensure_ascii=False 配合 encoding='utf-8' 写文件才能正确存中文。
  • 反序列化时要处理字段缺失（用 .get(k, 默认)）。
💡 课后练习
  • 给任务加「优先级」和「截止日期」字段。
  • 把工作区导出成人类可读的 Markdown。

main.py（核心）

import json

@dataclass
class Project:
    name: str
    tasks: list[dict] = field(default_factory=list)
    def add_task(self, title, done=False):
        if any(t["title"] == title for t in self.tasks):
            raise ValueError(f"已存在：{title}")
        self.tasks.append({"title": title, "done": done})

@dataclass
class Workspace:
    projects: dict[str, Project] = field(default_factory=dict)
    def save(self, path):
        path.write_text(json.dumps(self.to_dict(),
            ensure_ascii=False, indent=2), encoding="utf-8")
