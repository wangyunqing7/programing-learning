乘法表生成器

for 循环、嵌套循环、字符串对齐排版

📌 学习目标

  • 用 for 遍历 range()
  • 写嵌套循环（外层行、内层列）
  • 用 " ".join(...) 把列表拼成一行
  • 用 f"{x:>3}" 做右对齐排版

知识点

range：range(1, n+1) 生成 1 到 n（左闭右开，所以要 n+1）。

嵌套循环：外层循环每跑一次，内层循环完整跑一遍。乘法表里外层是「行号 a」，内层是「列号 b」。

join："分隔符".join(列表) 把多个字符串用分隔符拼起来，比手动加号拼接更高效、更干净。

⚠️ 易错点
  • 对齐宽度要按「最大结果」定，否则列会错位。
  • range(n) 是 0 到 n-1，不是 1 到 n。
💡 课后练习
  • 改成上三角：只输出 a ≤ b 的格子（加 if a <= b）。
  • 让 table(n) 返回字符串列表，由调用者决定怎么用。

main.py（核心）

def cell(a, b):
    return f"{a}×{b}={a*b:>3}"

def table(n):
    if n <= 0:
        raise ValueError("范围必须为正整数")
    rows = []
    for a in range(1, n + 1):
        line = "  ".join(cell(a, b) for b in range(1, n + 1))
        rows.append(line)
    return rows

▶ 运行结果（5×5）

1×1=  1  1×2=  2  1×3=  3  1×4=  4  1×5=  5
2×1=  2  2×2=  4  2×3=  6  2×4=  8  2×5= 10
3×1=  3  3×2=  6  3×3=  9  3×4= 12  3×5= 15
4×1=  4  4×2=  8  4×3= 12  4×4= 16  4×5= 20
5×1=  5  5×2= 10  5×3= 15  5×4= 20  5×5= 25
