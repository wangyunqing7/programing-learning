递归目录摘要

递归思想、嵌套结构遍历、缩进树

📌 学习目标

  • 理解递归的「基线 + 递推」两要素
  • 用 isinstance 区分叶子（int）和分支（dict）
  • 递归累加得到汇总
  • 用缩进打印树状结构

知识点

递归：函数调用自己。必须有「基线情况」（不再递归，如遇到 int 就返回）和「递推情况」（dict 就继续往里钻），否则无限递归。

分支/叶子：把目录树建模成「dict=分支，int=叶子（文件大小）」。isinstance(node, int) 判断是不是叶子。

累加：每个分支的统计 = 它所有子节点的统计之和，自然用递归表达。

⚠️ 易错点
  • 递归没有基线情况会无限循环直到栈溢出（RecursionError）。
  • 层级太深的真实目录用递归可能爆栈，工程上常用迭代。
💡 课后练习
  • 按扩展名分组统计（.py 多少、.md 多少）。
  • 把树导出成 JSON。

main.py（核心）

@dataclass
class Summary:
    files: int = 0
    size: int = 0
    def __add__(self, o): return Summary(self.files+o.files, self.size+o.size)

def summarize(node):
    if isinstance(node, int):
        return Summary(1, node)        # 叶子：基线
    total = Summary()
    for child in node.values():         # 分支：递推
        total = total + summarize(child)
    return total
