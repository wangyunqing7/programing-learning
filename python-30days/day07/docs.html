单位换算函数

函数定义、参数、返回值、函数表分发

📌 学习目标

  • 定义带参数和返回值的函数
  • 理解「先归一再转换」的统一换算思路
  • 用嵌套字典做换算系数表
  • 用异常处理不支持的类别/单位

知识点

归一化：选定一个基准单位（如长度的米）系数为 1.0，其它单位给出相对系数。任意两个单位互转 = 先乘源系数归一到基准，再除以目标系数。

函数表分发：用数据（字典）描述规则，而不是写一堆 if/elif，扩展新类别只需加一行数据。

def：def 名(参数) -> 返回类型:，参数可以有默认值。

⚠️ 易错点
  • 单位跨类别（如把米换算成千克）要报错而不是给错误答案。
  • 系数表写错一个数字，所有相关换算都会错。
💡 课后练习
  • 加一个「面积」类别（m²、km²、acre）。
  • 让 convert 支持「一次性输出所有单位」的批量换算。

main.py（核心）

CONVERSIONS = {
    "length": {"m": 1.0, "km": 1000.0, "cm": 0.01, "mile": 1609.34},
    "weight": {"kg": 1.0, "g": 0.001, "lb": 0.453592},
}

def convert(value, category, from_unit, to_unit):
    table = CONVERSIONS.get(category)
    if table is None:
        raise ValueError(f"不支持：{category}")
    if from_unit not in table or to_unit not in table:
        raise ValueError(f"单位不在 {category}")
    base = value * table[from_unit]
    return base / table[to_unit]

▶ 运行结果

长度换算：
  1 km = 0.6214 mile
  100 m = 328.0840 ft
重量换算：
  1 kg = 2.2046 lb
  1 m->kg: 单位不在 length 里：m/kg
