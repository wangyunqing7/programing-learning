list 增删查改、dataclass、JSON 持久化

📌 学习目标

  • 用 list 做 append / 删除（推导式过滤）
  • 用 @dataclass 定义数据类
  • 用 json 把对象存盘、读盘
  • 用 @classmethod 做工厂方法

知识点

dataclass：用 @dataclass 装饰一个类，Python 自动生成 __init__、__repr__ 等，省去手写样板。

可变默认值：tasks: list = [] 是陷阱（所有实例共享同一个列表）！必须用 field(default_factory=list)。

JSON 序列化：asdict(obj) 把 dataclass 转成字典，json.dumps 再转成字符串；读回来用 Task(**d) 重建。

⚠️ 易错点
  • list.append() 返回 None，不能 tasks = tasks.append(x)。
  • 可变默认值要用 field(default_factory=...)。
  • 读盘时文件可能不存在，load 要先 path.exists() 判断。
💡 课后练习
  • 加一个 filter_by_priority(n) 方法。
  • 把存储改成 CSV 而不是 JSON。

main.py（核心）

@dataclass
class Task:
    title: str
    done: bool = False
    priority: int = 2

@dataclass
class TaskManager:
    tasks: list[Task] = field(default_factory=list)

    def add(self, title, priority=2):
        if not title.strip():
            raise ValueError("标题不能为空")
        self.tasks.append(Task(title, priority=priority))

    def complete(self, title):
        for t in self.tasks:
            if t.title == title:
                t.done = True
                return
        raise KeyError(f"找不到：{title}")

    def save(self, path):
        path.write_text(json.dumps(
            [asdict(t) for t in self.tasks], ensure_ascii=False, indent=2),
            encoding="utf-8")
