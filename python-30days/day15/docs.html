列表推导统计

推导式、过滤、海象运算符、statistics

📌 学习目标

  • 写列表/字典/集合推导式
  • 在推导式里加 if 过滤
  • 用海象运算符 := 复用表达式结果
  • 用 statistics 模块算均值/中位数/标准差

知识点

推导式：[表达式 for x in 列表 if 条件] 一行生成新列表，比 for+append 简洁。

海象 :=：在表达式里赋值。... if (p := s['python']) >= 60 先把分数存到 p，后面直接用，避免重复索引。

statistics：mean/median/pstdev 算常用统计量。空列表会抛 StatisticsError。

⚠️ 易错点
  • 推导式太长会难读，超过一行考虑改回 for 循环。
  • 空数据集调 statistics 会报错，要先判空。
💡 课后练习
  • 用集合推导找出「各科都及格」的人。
  • 用字典推导生成「名字→等级」映射。
