安全除法

异常分类、自定义异常、match/case

📌 学习目标

  • 用 try/except 捕获不同类型异常
  • 继承 ValueError 定义业务异常
  • 用 if/elif 按运算符分发
  • 理解异常的继承关系

知识点

异常分类：不同错误用不同异常类，调用方可以分别 catch。除零用 ZeroDivisionError，业务非法用自定义 InvalidInputError。

自定义异常：class MyError(ValueError): 继承一个合适的父类，既能被精确 catch，也能被父类 catch。

运算分发：用 if/elif 按运算符字符串走不同分支，逻辑清晰、所有版本通用。

⚠️ 易错点
  • 别用 except Exception 一把抓，会吞掉 bug，难以排查。
  • 自定义异常要继承合适的父类，否则调用方难精确捕获。
💡 课后练习
  • 加「幂运算 ^」和「取模 %」支持。
  • 让 safe_eval 支持带括号的表达式。

main.py（核心）

class InvalidInputError(ValueError):
    pass

def safe_divide(a, b):
    if b == 0:
        raise ZeroDivisionError(f"除数为 0：{a}/{b}")
    return a / b

def safe_eval(expr):
    parts = expr.split()
    a, op, b = float(parts[0]), parts[1], float(parts[2])
    if op == "/":    return safe_divide(a, b)
    elif op == "sqrt": return safe_sqrt(a)
    else: raise InvalidInputError(f"不支持：{op}")
