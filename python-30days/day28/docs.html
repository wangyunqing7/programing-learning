单元测试示例

unittest、断言、测试用例

📌 学习目标

  • 用 unittest.TestCase 写测试
  • 掌握 assertEqual/True/Raises 等断言
  • 一个被测函数配多个测试用例（正常 + 边界）
  • 用 TestLoader 加载运行

知识点

为什么测试：测试是把「期望行为」固化成代码，改代码时自动验证没改坏。

断言：assertEqual(a, b) 判相等，assertTrue(x) 判真，assertRaises(Err) 判抛异常。失败会报告，不中断其他用例。

测试隔离：每个 test_ 方法独立，互不影响。

⚠️ 易错点
  • 测试名必须以 test_ 开头才会被发现。
  • 断言要针对行为（输出），不要针对实现细节。
💡 课后练习
  • 给 is_palindrome 加中文测试用例。
  • 用 setUp 准备共享测试数据。

main.py（核心）

import unittest

def is_palindrome(s):
    s = "".join(c.lower() for c in s if c.isalnum())
    return s == s[::-1]

class TestPalindrome(unittest.TestCase):
    def test_simple(self):
        self.assertTrue(is_palindrome("level"))
    def test_phrase(self):
        self.assertTrue(is_palindrome("A man a plan a canal Panama"))
    def test_negative(self):
        self.assertFalse(is_palindrome("python"))
