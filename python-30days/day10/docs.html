临时笔记文件

pathlib、文件读写、临时目录

📌 学习目标

  • 用 Path 拼接路径、读写文件
  • 用 read_text/write_text 一步读写
  • 用 glob 匹配文件
  • 用 TemporaryDirectory 自动清理

知识点

Path：Path(a) / b 用除号拼路径，跨平台都安全（自动处理分隔符）。

读写：p.read_text(encoding='utf-8') 和 p.write_text(s) 比手写 open/close 简洁。

临时目录：with TemporaryDirectory() as d: 退出 with 块时自动删除整个目录，适合做练习不污染磁盘。

⚠️ 易错点
  • 读写中文要指定 encoding='utf-8'，否则 Windows 默认 GBK 会乱码。
  • 退出 TemporaryDirectory 的 with 块后目录已被删，不能再访问。
💡 课后练习
  • 加 append 方法（用 open('a') 追加）。
  • 用 stat().st_size 显示每个笔记的大小。

main.py（核心）

from pathlib import Path
from tempfile import TemporaryDirectory

@dataclass
class Notebook:
    root: Path
    def write(self, name, content):
        path = self.root / f"{name}.md"
        path.write_text(content, encoding="utf-8")
        return path
    def read(self, name):
        path = self.root / f"{name}.md"
        if not path.exists():
            raise FileNotFoundError(f"没有：{name}")
        return path.read_text(encoding="utf-8")
    def list_pages(self):
        return sorted(self.root.glob("*.md"))
