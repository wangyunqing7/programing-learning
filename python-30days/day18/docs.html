日期计划表

datetime、timedelta、日期运算

📌 学习目标

  • 用 date/timedelta 做日期运算
  • 用 weekday() 取星期几
  • 计算两个日期的间隔（倒计时）
  • 用 .isoformat() 标准化输出

知识点

timedelta：表示时间差，date + timedelta(days=n) 得到 n 天后的日期。

weekday()：返回 0（周一）到 6（周日），显示时要 +1。

日期相减：date1 - date2 得到 timedelta，.days 取天数。未来日期相减为正，过去为负。

⚠️ 易错点
  • weekday() 返回 0-6，不是 1-7。
  • 日期相减结果是 timedelta 对象，要 .days 才能拿到整数。
💡 课后练习
  • 跳过周末排期（只在工作日安排任务）。
  • 加上具体时刻（用 datetime 而非 date）。

main.py（核心）

from datetime import date, timedelta

@dataclass
class Plan:
    start: date
    tasks: list[str]
    def schedule(self):
        return [(self.start + timedelta(days=i), t)
                for i, t in enumerate(self.tasks)]
    def countdown(self, today, target):
        return (target - today).days
