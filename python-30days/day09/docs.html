文本统计器

字符串处理、Counter、Top-N

📌 学习目标

  • 用 str.replace/split/lower 清洗文本
  • 用 collections.Counter 统计词频
  • 用 most_common(n) 取 Top-N
  • 用 dataclass 封装统计结果

知识点

清洗：先把标点替换成空格，再 split() 分词，避免 hello, 和 hello 被当成两个词。

Counter：Counter(list) 自动数每个元素出现次数，.most_common(5) 直接取前 5。

停用词：常见词（the/a/is）信息量低，统计前先过滤掉。

⚠️ 易错点
  • 不清洗标点会导致同一单词被统计成多个。
  • 空文本会让 max(words, key=len) 报错，要先判断非空。
💡 课后练习
  • 支持中文：按字（而不是按空格分词）统计。
  • 把结果存成 CSV 词频表。

main.py（核心）

from collections import Counter
STOP_WORDS = {"the", "a", "an", "is", "to", "of"}

def analyze(text, top_n=5):
    if not text.strip():
        raise ValueError("文本为空")
    cleaned = text
    for ch in ",.;:!?\"'()[]":
        cleaned = cleaned.replace(ch, " ")
    words = [w.lower() for w in cleaned.split() if w]
    words = [w for w in words if w not in STOP_WORDS]
    return Counter(words).most_common(top_n)
