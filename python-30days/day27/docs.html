SQLite 记账

sqlite3、建表、参数化查询、聚合

📌 学习目标

  • 用 sqlite3 操作内存数据库
  • 写 CREATE TABLE / INSERT
  • 用 ? 参数化查询防注入
  • 用 SUM/GROUP BY/CASE 聚合

知识点

关系型：数据存表里（行=记录，列=字段），SQLite 是 Python 内置的轻量数据库，:memory: 表示建在内存里（退出即消失）。

参数化：execute(sql, (a,b)) 用 ? 占位，绝不用字符串拼接 SQL——后者有注入风险。

聚合：SUM(CASE WHEN kind='income' THEN amount ELSE -amount END) 一句算出收支相抵的余额。

⚠️ 易错点
  • 字符串拼 SQL 是安全大忌，永远用 ? 参数化。
  • fetchone() 对空表返回 None，取 [0] 前要判断。
💡 课后练习
  • 给每笔交易加日期，按月汇总。
  • 加「按类别删除」功能。

main.py（核心）

import sqlite3

class Ledger:
    @classmethod
    def open(cls):
        conn = sqlite3.connect(":memory:")
        conn.execute("CREATE TABLE txns (id INTEGER PRIMARY KEY,"
                     "kind TEXT, category TEXT, amount REAL)")
        return cls(conn)
    def add(self, kind, category, amount):
        self.conn.execute(
            "INSERT INTO txns VALUES (NULL,?,?,?)", (kind,category,amount))
    def balance(self):
        return self.conn.execute(
            "SELECT SUM(CASE kind WHEN 'income' THEN amount ELSE -amount END)"
            " FROM txns").fetchone()[0] or 0.0
