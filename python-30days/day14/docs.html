图书 dataclass

dataclass、对象列表、借还库存

📌 学习目标

  • 用 @dataclass + 默认值定义图书
  • 管理「馆藏 vs 可借」两个数量
  • 实现借阅、归还、搜索
  • 用字典做书名→书的索引

知识点

两个计数：copies 是馆藏总数（不变），borrowed 记录借出数，可借 = copies − borrowed。

__str__：定义 __str__ 让 print(book) 输出友好格式，而不是 <Book object at 0x...>。

索引：用 dict[书名]=Book 做索引，查找比遍历 list 快。

⚠️ 易错点
  • 借出时要检查「可借数 > 0」，否则超借。
  • 馆藏数增加时，已有的书要累加而不是覆盖。
💡 课后练习
  • 记录「借阅者是谁」。
  • 加「预约」功能：书全借出时可排队。

main.py（核心）

@dataclass
class Book:
    title: str
    author: str
    copies: int = 1
    def __str__(self):
        return f"《{self.title}》-{self.author} (馆藏 {self.copies})"

@dataclass
class Library:
    books: dict[str, Book] = field(default_factory=dict)
    borrowed: dict[str, int] = field(default_factory=dict)
    def borrow(self, title):
        book = self.books.get(title)
        available = book.copies - self.borrowed.get(title, 0)
        if available <= 0:
            raise ValueError(f"《{title}》已全部借出")
        self.borrowed[title] = self.borrowed.get(title, 0) + 1
