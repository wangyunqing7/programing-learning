路径整理器

pathlib 路径分析（不碰磁盘）

📌 学习目标

  • 区分 PurePath（纯分析）和 Path（碰磁盘）
  • 取路径的各部分：suffix/name/parent/anchor
  • 按分隔符判断 Windows/Posix 路径
  • 统计扩展名分布

知识点

PurePath：只做字符串层面的路径分析，不访问文件系统，安全且跨平台。PureWindowsPath 认反斜杠，PurePosixPath 认正斜杠。

路径属性：.suffix 扩展名、.name 文件名、.parent 父目录、.anchor 盘符/根。

拆分：分析路径就是把这些属性拼回来理解它的结构。

⚠️ 易错点
  • PurePath 不检查文件是否存在，要判断存在性用 Path.exists()。
  • 扩展名大小写不统一，统计前要 .lower()。
💡 课后练习
  • 把重复的文件名标出来。
  • 识别路径里的无效字符。

main.py（核心）

from pathlib import PurePath, PureWindowsPath, PurePosixPath

def normalize(raw):
    return PureWindowsPath(raw) if "\\" in raw else PurePosixPath(raw)

def summarize(paths):
    exts = {}
    for raw in paths:
        p = normalize(raw)
        ext = p.suffix.lower() or "(无)"
        exts[ext] = exts.get(ext, 0) + 1
        print(p.anchor, p.parent, p.name)
