支出统计器

数据聚合、分类汇总、报表

📌 学习目标

  • 用 defaultdict 做分组聚合
  • 按多个维度（类别、日期）汇总
  • 用字符串画简易柱状图
  • 计算占比百分比

知识点

defaultdict：defaultdict(float) 访问不存在的键自动返回 0.0，省去「先判断在不在」的啰嗦。

聚合：遍历每条支出，agg[category] += amount 累加到对应类别。

柱状图：'█' * int(pct/5) 用字符重复画条，pct 越高条越长。

⚠️ 易错点
  • defaultdict 访问不存在的键会「创建」它，遍历时要注意别误增键。
  • 百分比相加可能不等于 100（四舍五入），正常现象。
💡 课后练习
  • 按周（不是按天）聚合。
  • 找出「花费最多的那一天」。

main.py（核心）

from collections import defaultdict

def by_category(items):
    agg = defaultdict(float)
    for e in items:
        agg[e.category] += e.amount
    return dict(sorted(agg.items(), key=lambda kv: -kv[1]))

def report(items):
    total = sum(e.amount for e in items)
    for cat, amt in by_category(items).items():
        pct = amt / total * 100
        print(f"{cat:<8} {amt:>8.2f} {pct:5.1f}% {'█'*int(pct/5)}")
