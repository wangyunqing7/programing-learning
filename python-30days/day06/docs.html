联系人字典

dict 键值结构、模糊搜索、排序

📌 学习目标

  • 用 dict 存键值对，掌握增删查
  • 用 .get() 避免 KeyError
  • 用 .lower() 做大小写不敏感的模糊匹配
  • 用 sorted(key=...) 排序

知识点

dict：键值对的集合，键唯一。增删改分别是 d[k]=v、del d[k]、d[k]=new。

安全取值：d.get(k) 找不到返回 None，不会报错；d[k] 找不到会 KeyError。

模糊搜索：把关键字和被搜索文本都 .lower()，再用 in 判断包含。

⚠️ 易错点
  • 用 d[k] 访问不存在的键会 KeyError，查询场景改用 .get()。
  • 重复 add 同一个名字要主动报错，否则会覆盖旧数据。
💡 课后练习
  • 加一个「按手机号搜索」的方法。
  • 把联系人存成 JSON 文件实现持久化。

main.py（核心）

@dataclass
class Contact:
    name: str
    email: str
    phone: str = ""

@dataclass
class AddressBook:
    contacts: dict[str, Contact] = field(default_factory=dict)

    def add(self, c):
        if c.name in self.contacts:
            raise KeyError(f"已存在：{c.name}")
        self.contacts[c.name] = c

    def search(self, keyword):
        kw = keyword.lower()
        return [c for c in self.contacts.values()
                if kw in c.name.lower() or kw in c.email.lower()]
