学习仪表盘

综合复盘、数据汇总、报表

📌 学习目标

  • 汇总 30 天学过的所有知识点
  • 用 dataclass + 列表组织结构化数据
  • 按周分组生成报表
  • 回顾整门课程的脉络

知识点

数据建模：把每天抽象成 Topic(day, title, skill)，30 个 Topic 按周分组成字典，便于统计和展示。

汇总：sum(len(v) for v in WEEKLY.values()) 算总天数，{t.skill for ...} 用集合去重算知识点数。

复盘价值：把零散的 30 天串成一条线（基础→数据结构→函数→文件→OOP→标准库→综合），看到自己的成长曲线。

⚠️ 易错点
  • 数据写死在代码里，如果要展示真实进度需对接外部数据。
  • 集合去重时，描述相同的知识点会被合并。
💡 课后练习
  • 把仪表盘数据存成 JSON 文件。
  • 加「掌握程度」自评字段并可视化。

main.py（核心）

@dataclass
class Topic:
    day: int
    title: str
    skill: str

WEEKLY = {
    "第1周": [Topic(1,"Hello CLI","print/变量"), ...],
    "第4周": [Topic(27,"SQLite","sqlite3"), Topic(30,"仪表盘","综合")],
}

def dashboard():
    total = sum(len(v) for v in WEEKLY.values())
    for week, topics in WEEKLY.items():
        print(f"=== {week}（{len(topics)} 天）===")
