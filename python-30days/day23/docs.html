排序和搜索

sorted 的 key、二分查找 bisect

📌 学习目标

  • 用 sorted(key=...) 自定义排序规则
  • 用元组做 key 实现多关键字排序
  • 用 bisect 在有序列表里二分查找
  • 理解负号实现降序的技巧

知识点

key 函数：sorted(lst, key=lambda x: x.score) 按 score 排，原列表不变。

多关键字：key 返回元组 (-score, name)，先按分数降序（负号），同分按名字升序。

bisect：bisect_right(有序列表, x) 返回 x 应插入的位置，底层是二分，O(log n)。

⚠️ 易错点
  • 用负号降序只对数字有效，字符串降序要用 reverse=True。
  • bisect 要求列表已经有序，否则结果无意义。
💡 课后练习
  • 实现「按姓名长度」排序。
  • 手写一个二分查找（不用 bisect）。

main.py（核心）

import bisect

def by_score_then_name(students):
    return sorted(students, key=lambda s: (-s.score, s.name))

def rank_of(sorted_scores, target):
    # 在升序列表里找 target 的名次：比它高的有多少个
    return len(sorted_scores) - bisect.bisect_right(sorted_scores, target) + 1
