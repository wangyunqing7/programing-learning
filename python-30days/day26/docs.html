配置读取器

configparser、默认值、类型校验

📌 学习目标

  • 用 ConfigParser 读写 INI 配置
  • 用 read_dict 预设默认值
  • 用 getint/getboolean 做类型转换
  • 校验必填项

知识点

INI 结构：分节 [section]，每节若干 key = value。ConfigParser 把它读成嵌套字典。

默认值：先 read_dict(DEFAULTS) 塞默认，再 read(文件) 覆盖，用户没填的项自动有默认值。

类型方法：所有值读出来是字符串，getint 转 int，getboolean 把 true/yes/1 转 bool。

⚠️ 易错点
  • 不填默认值时，用户漏填会 KeyError。
  • getboolean 只认 true/false/yes/no/1/0，写别的会报错。
💡 课后练习
  • 加 [logging] 节含 level。
  • 支持从环境变量覆盖配置。

main.py（核心）

import configparser
DEFAULTS = {"app": {"name":"MyApp","debug":"false"},
           "db": {"host":"localhost","port":"5432"}}

def load(path):
    cfg = configparser.ConfigParser()
    cfg.read_dict(DEFAULTS)
    if not cfg.read(path):
        raise FileNotFoundError(f"不存在：{path}")
    if not cfg.get("db", "host"):
        raise ValueError("db.host 不能为空")
    return cfg
