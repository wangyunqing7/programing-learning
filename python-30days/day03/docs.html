成绩判断器

if/elif/else 多分支、加权统计

📌 学习目标

  • 用 if/elif/else 写多分支判断
  • 理解分支「从上到下短路」的特性
  • 用字典做权重，求加权总分
  • 处理非法输入（负分、权重和≠1）

知识点

分支顺序：elif score >= 90 ... elif score >= 80 必须从高到低写。因为一旦某个条件成立，后面的都不会再判断——如果先写 >=60，那 95 分也会被归到「及格」。

加权：每科乘以权重再相加。权重之和应该等于 1，否则结果失真。浮点数比较不能直接用 ==，要用容差 abs(x - 1.0) < 1e-6。

⚠️ 易错点
  • 分支条件写反方向，导致分类全错。
  • 权重和不为 1 时加权分没有意义，要主动校验。
💡 课后练习
  • 加一个「缺考」状态：某科为 None 时按 0 分算并标注。
  • 让 grade_of 支持 A/B/C/D 字母等级。

main.py（核心）

def grade_of(score):
    if score >= 90:   return "优秀"
    elif score >= 80: return "良好"
    elif score >= 60: return "及格"
    elif score >= 0:  return "不及格"
    else: raise ValueError(f"非法分数：{score}")

def weighted_total(scores, weights):
    if abs(sum(weights.values()) - 1.0) > 1e-6:
        raise ValueError("权重之和必须等于 1")
    return sum(scores.get(s, 0) * w for s, w in weights.items())
