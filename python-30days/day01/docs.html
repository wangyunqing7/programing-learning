Hello CLI

print、变量与字符串、脚本入口 __name__

📌 学习目标

  • 用 print() 向终端输出文本
  • 理解变量与字符串拼接 / f-string
  • 看懂 if __name__ == "__main__": 的作用
  • 从命令行参数 sys.argv 读取输入

知识点

print 是最基础的输出函数，可以接收多个参数，默认用空格分隔、以换行结尾。

变量 是给数据起的名字。字符串可以用 .format() 或 f"..." 嵌入变量。

__name__：每个 Python 文件被执行时，__name__ 等于 "__main__"；被别人 import 时则等于模块名。所以 if __name__ == "__main__": 下的代码只在「直接运行」时才执行，被导入时不执行。

sys.argv 是命令行参数列表，argv[0] 是脚本名本身，argv[1] 才是第一个参数。

⚠️ 易错点
  • sys.argv 没传参数时直接取 argv[1] 会 IndexError，所以要先判断长度。
  • 字符串 .center(n) 按字符数对齐，中文占一个位置但显示宽两个，名片里需要手动补空格。
💡 课后练习
  • 在 GREETINGS 里加一种语言（比如 "ko"），重新运行。
  • 把名片改成英文版，观察对齐效果。
  • 运行 python main.py 你的名字，看 sys.argv 如何生效。

main.py

from __future__ import annotations
import sys

GREETINGS: dict[str, dict[str, str]] = {
    "zh": {"hello": "你好，{name}！", "lang_name": "中文"},
    "en": {"hello": "Hello, {name}!", "lang_name": "English"},
    "ja": {"hello": "こんにちは、{name}さん！", "lang_name": "日本語"},
}

def pick_name(argv: list[str]) -> str:
    if len(argv) >= 2 and argv[1].strip():
        return argv[1].strip()
    return "Python 学习者"

def greet(name: str, lang: str = "zh") -> str:
    template = GREETINGS.get(lang, GREETINGS["zh"])
    return template["hello"].format(name=name)

def main() -> None:
    name = pick_name(sys.argv)
    print(greet(name))
    for lang in ("zh", "en", "ja"):
        print(greet(name, lang))

if __name__ == "__main__":
    main()
